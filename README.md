# Sales Trend Analysis & Forecasting System

## Description
This project provides a comprehensive system for analyzing sales trends and forecasting future sales using historical data. Utilizing the Superstore Sales Dataset as an example, it performs crucial steps including data loading, cleaning, exploratory data analysis (EDA), visualization, and predictive modeling. The system aims to empower businesses with actionable insights into their sales performance and potential future trends, enabling data-driven decision-making.

The `sales_analysis.py` script generates key business metrics, creates an interactive visualization dashboard, and produces a sales forecast using linear regression.

## Tech Stack
The project is built primarily with Python and leverages several powerful libraries for data science and visualization:

*   **Python**: The core programming language.
*   **Pandas**: For robust data manipulation and analysis.
*   **NumPy**: For numerical operations, especially with arrays.
*   **Matplotlib**: For creating static, animated, and interactive visualizations.
*   **Seaborn**: Built on Matplotlib, providing a high-level interface for drawing attractive and informative statistical graphics.
*   **Scikit-learn**: For machine learning, specifically `LinearRegression` for forecasting and `mean_absolute_error`, `mean_squared_error` for model evaluation.

## Installation

To set up the project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/annu0603/Sales-Trend-Analysis-Forecasting-System.git
    cd Sales-Trend-Analysis-Forecasting-System
    ```

2.  **Create a virtual environment (recommended):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```

3.  **Install the required Python packages:**
    ```bash
    pip install pandas numpy matplotlib seaborn scikit-learn
    ```

4.  **Obtain the Superstore Sales Dataset:**
    Download the `Sample - Superstore.csv` file. You can typically find this dataset on Kaggle or other data science platforms. Place the downloaded CSV file in the root directory of the cloned repository. Ensure its name matches `'Sample - Superstore.csv'` or update the `filepath` variable in `sales_analysis.py` accordingly.

## Usage

To run the sales analysis and forecasting system:

1.  Ensure you have followed all steps in the [Installation](#installation) section, especially placing the `Sample - Superstore.csv` file in the project's root directory.

2.  Execute the `sales_analysis.py` script:
    ```bash
    python sales_analysis.py
    ```

Upon execution, the script will:
*   Print various data cleaning steps and key business metrics to the console.
*   Display visualizations (monthly sales, sales by category/region, seasonal patterns) in an interactive matplotlib window.
*   Save a comprehensive dashboard as `superstore_dashboard.png`.
*   Display the sales forecast visualization (historical trend, model performance, and future forecast) in an interactive matplotlib window.
*   Save the forecast chart as `superstore_forecast.png`.
*   Print a summary of the forecast to the console.

## Configuration

The `sales_analysis.py` script has a few configurable parameters within its `if __name__ == "__main__":` block:

*   **`filepath`**: The path to your sales dataset.
    ```python
    filepath = 'Sample - Superstore.csv' # Update this if your file has a different name or path
    ```
    If your dataset file is named differently (e.g., `superstore_data.csv`), update this variable:
    ```python
    filepath = 'superstore_data.csv'
    ```

*   **`forecast_days`**: The number of days into the future for which the sales forecast will be generated. It defaults to 90 days.
    ```python
    model, forecast = forecast_sales(df, forecast_days=90) # Change 90 to your desired forecast horizon
    ```
    To forecast for, say, 180 days, modify the line:
    ```python
    model, forecast = forecast_sales(df, forecast_days=180)
    ```
